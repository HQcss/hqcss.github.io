HQ.css và HQ.js được phát triển bởi HQ-Multimedia là file hỗ trợ nhanh trong lập trình Front-End website
File được viết sẵn các cấu trúc js và css kết hợp với nhiều plugin sẵn có như bootstrap,video-js,animate,isotope... tuỳ vào file thiết kế thực tế có thể tuỳ chỉnh theo dự án với setting-hq.css
Với mong muốn việc làm web trở nên nhanh chóng chỉ cần copy-paste và thay đổi các class
